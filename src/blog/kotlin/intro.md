# Kotlin

## 为什么使用 Kotlin

Kotlin 是一种静态类型的编程语言， 它是由 JetBrains 开发的， 它的目标是成为一种通用的、 静态类型的、 并发性的、 函数式的、 跨平台的编程语言。

Kotlin 可以在 JVM、 JS、 Native 等平台上运行， 它的语法简洁、 表达力强、 类型安全， 同时也支持面向对象、 函数式、 并发等编程范式。

## Java 交互

Kotlin 与 Java 可以无缝交互， 这意味着你可以在 Kotlin 中调用 Java 代码， 也可以在 Java 中调用 Kotlin 代码。
